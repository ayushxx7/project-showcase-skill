## 🚀 Why it Exists

Building a project is 90% of the work, but showing it to the world is the other 90%. 

**The Problem**: Taking manual screenshots, cropping them, and organizing them into a README is tedious. 
**The Solution**: `project-showcase-skill` automates this process, giving you high-quality visuals and documentation in seconds.

---

## 🎬 Automated UI Capture

The skill uses a generalized Playwright script to capture high-resolution screenshots of your application. It handles hydration waits, responsive design, and complex UI interactions.

## 🛠️ Core Workflows

1.  **Initial Analysis**: Identifies project type and key routes.
2.  **Automated Capture**: Generates and executes a Playwright script for UI capture.
3.  **README Documentation**: Builds a professional README using high-converting templates.
4.  **Social Media Summary**: Distills the project into value propositions for LinkedIn or Twitter.

## ⚡ Manual Setup (Development)

If you wish to run the scripts directly or contribute:

```bash
# 1. Install dependencies
uv pip install playwright
uv run playwright install chromium

# 2. Run capture script
uv run python scripts/capture.py --url http://localhost:8501 --dir showcase/
```
