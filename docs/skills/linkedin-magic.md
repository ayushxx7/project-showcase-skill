# LinkedIn Showcase Magic Skill 🚀

This skill is your personal PR agent. It transforms raw code into professional social currency by automating the "Documentation -> Hype" pipeline.

## Core Workflows

### 1. The "Karma Farm" Prep
- **Goal**: Ensure the project is visually stunning and architecturally clear before posting.
- **Action**: Invokes `project-showcase` to update the architecture diagram, health score, and UI screenshots.

### 2. High-Impact Content Generation
- **Variants**:
    - **🚀 The Viral Hype**: Short, punchy, high-conversion.
    - **🏗️ The Deep Dive**: Technical architecture and tech stack logic.
    - **📈 The Health Audit**: Focuses on trust and quality metrics.
    - **💡 The Story**: Problem/Solution narrative.

### 3. Visual Asset Bundling
- **Goal**: Package the "Showcase Gallery" for easy upload.
- **Action**: Copies the best assets (Landing, Dashboard, Architecture) into a local `linkedin_launch/` folder.

### 4. Hashtag & Engagement Strategy
- **Strategy**: Inject trending hashtags like #BuildingInPublic #AIAgents #GeminiCLI #ShowcaseSkill.
- **Engagement**: Provides a "Call to Action" (CTA) strategy for comments.
