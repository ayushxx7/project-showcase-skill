# Responsive Capture 📱

In the age of mobile-first development, a single desktop screenshot isn't enough. The Project Showcase Skill automatically captures your UI across multiple screen sizes.

## 📱 Supported Viewports

The `capture.py` script comes pre-configured with industry-standard viewports:
- **Desktop**: 1920 x 1080
- **Tablet**: 768 x 1024
- **Mobile**: 375 x 667 (iPhone-scale)

## Why it Matters

1. **Portfolio Quality**: Shows recruiters and users that you've considered responsive design.
2. **Feature Discovery**: Some features (like mobile menus) are only visible on smaller screens.
3. **Visual Verification**: Ensures your layout doesn't break under different conditions.

## Customizing Viewports

While the skill handles the defaults, you can ask for specific captures:
- *"Capture the mobile login screen."*
- *"Take a 4K resolution screenshot of the dashboard."*
