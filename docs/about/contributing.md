# Contributing

Contributions welcome — bug fixes, features, docs improvements.

## Setup

```bash
git clone https://github.com/ayushxx7/project-showcase-skill
cd project-showcase-skill
./scripts/setup.sh
```

## Testing

```bash
python3 -m pytest tests/
```

## Workflow

1. Branch from `main`
2. Add tests for new logic
3. Update docs if adding/changing features
4. Open a PR
